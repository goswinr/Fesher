Fesher is a library for colorful console output in the F# scripting editor [Fesh](https://github.com/goswinr/Fesh)

### Usage

```fsharp
open Fesher

Printfn.red "The red answer is %s!" 42
Printfn.green "The green answer is %s!" 42

```
